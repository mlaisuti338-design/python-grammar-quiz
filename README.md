# 🧠 Python 文法 穴埋めクイズ（Supabase連携）

Python 初学者向けの文法穴埋めクイズアプリです。  
Streamlit を用いて UI を作成し、Supabase のデータベースに回答履歴を保存することで、学習データが永続的に残るようにしています。

## 🔗 アプリURL

以下のURLから、誰でもアプリを試すことができます。

👉 https://あなたの-streamlit-アプリのURL

---

## ✨ 主な機能

- Python 文法に関する **穴埋めクイズ**
- ヒント表示機能（段階的に表示）
- 回答の **正誤判定と解説表示**
- **前の問題 / 次の問題** に移動可能
- 回答履歴を **Supabase に保存（永続化）**
- 不正解だった問題のみを出題する **復習モード**

## 工夫した点
- 文法理解を重視した穴埋め形式
- 間違えた問題を復習できる機能
- 問題データを別ファイルで管理

## 🛠 使用技術

- Python
- Streamlit
- Supabase（PostgreSQL）
- GitHub / Streamlit Cloud


## 📌 補足

- Supabase の接続情報は Streamlit の `secrets` 機能を使って管理しています
- sqlite3 などの簡易データベースは使用していません
- Supabase の Free プランで動作します

## 📚 目的

Python 初学者が、文法を「読むだけ」でなく  
**実際に考えながら学習できること**を目的としたアプリです。
大変でした。
